Aleysha Bradbury's Portfolio

Link to repo to download: Honours Project Game

In this project I created a turn/grid based game where the player has to move across the grid, talking to the blue npcs and completing quests for them.
This was a solo project and was made using SFML.
For this project I implemented the A* algorithm for pathfinding across the grid and I also had to create an algorithm for procedurally generating the quests.

Link to repo to download: Card Game

In this project I made a basic card game where the player plays their cards then the enemy will hit the player on their turn. The player has to press R on their keyboard to end their turn. This is a left over from when it meant R for Reset Hand and I never changed it.
This was a solo project and was made using SFML.

Link to game: HTML Game

This project is the first time I'm messing around with HTML/JavaScript. The game is a very early version of an incremental/idle game cause those work well for what I was trying to learn.
The player starts out with one button to press and two tabs and as they play the game they unlock more options for buttons.
The player can also change the colour of all of the buttons which I added 1) because of colour blind people who might look at it and 2) because I wanted to figure out how to change the style of something using JavaScript.
